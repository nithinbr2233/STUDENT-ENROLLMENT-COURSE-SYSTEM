# STUDENT-ENROLLMENT-COURSE-SYSTEM
 Project Description
This project is a database management system designed to manage students, courses, enrollments, exams, and results.

##  Objectives
- Manage student and course data
- Track enrollments
- Store exam details
- Analyze student performance

##  Technologies Used
- MySQL
- SQL Queries

##  Database Tables
- Students
- Courses
- Enrollment
- Exams
- Results

##  Features
- Many-to-Many relationship between Students and Courses
- One-to-Many relationship between Students and Exams
- Performance tracking using marks and grades

##  How to Run
1. Open MySQL
2. Run schema.sql
3. Run inserts.sql
4. Run queries.sql
